Design Concept: Flight Status Board

The goal of this design is to immerse the user in the airport environment, making them feel like a traveler checking the departures board right before a flight. The result table looks like the destination and arrival time. Every element is chosen to replicate that pre-travel excitement.

  • Color Scheme: Inspired by classic airport's display system, the color scheme uses yellow to highlight the selected "departure" currency against a dark-grey background. This directly mimics the look of a flight status board.
  • Wording: The language used throughout the interface reinforces the travel theme. Phrases like "Choose Your Departure" and the "✈️ TAKE OFF!" button make the process feel less like a calculation and more like the start of a journey.
  • Dynamic Background: A looping video of a real airport departures board serves as the full-screen background. This creates an authentic and active environment, placing the user directly in the bustling atmosphere of a terminal.
  • Results Layout: The results table is structured to mirror an arrivals board. The 'Currency' on the left acts as the 'Destination', and the 'Amount' on the right represents key flight information.
